# ride_arm

## Includes

- Vue.js (class-component + property-decorator)
- Vuex (vuex-class)
- Typescript
- Ramda
- Pug
- Sass
- Eslint
- Prettier
- Typedoc
- Jest
- Docker

## Project setup

```sh
yarn
```

## Compiles and hot-reloads for development

```sh
yarn run serve
```

## Compiles and minifies for production

```sh
yarn run build
```

## Lints and fixes files

```sh
yarn run lint
```

## Run your unit tests

```sh
yarn run test
```

## Build container and run

```sh
docker-compose up --build
```

## Just run container

```sh
docker-compose up
```

## Start fake api

```sh
json-server --watch db.json
```
